# FinAI 

FinAI is an Android-based personal finance management application that helps users track expenses, monitor spending habits, and gain financial insights through an intelligent dashboard. The application can analyze transaction SMS messages from payment platforms and banks to automatically record expenses and provide users with a complete overview of their financial activities.

## Features

* Automatic expense tracking from transaction SMS messages
* Support for Google Pay, PhonePe, Paytm, and bank transaction alerts
* Smart expense categorization
* Personal finance dashboard
* Monthly and yearly spending analysis
* Budget planning and monitoring
* Transaction history management
* Financial insights and spending trends
* User-friendly mobile interface

## Dashboard Analytics

The dashboard provides:

* Total expenses overview
* Monthly spending summary
* Category-wise expense breakdown
* Recent transactions
* Spending trend visualization
* Budget utilization tracking
* Financial activity reports

## Project Objectives

The primary objective of FinAI is to simplify personal finance management by automatically capturing financial transactions and presenting meaningful insights through an interactive dashboard.

## Key Functionalities

### Transaction Tracking

* Reads financial transaction SMS messages
* Detects debit and credit transactions
* Extracts transaction amount, date, and merchant details
* Stores transaction records automatically

### Expense Management

* Categorizes expenses automatically
* Allows manual expense entry
* Tracks daily, weekly, and monthly spending
* Maintains complete transaction history

### Analytics Dashboard

* Visual spending reports
* Category-wise analysis
* Budget monitoring
* Expense trends and patterns
* Financial summaries
